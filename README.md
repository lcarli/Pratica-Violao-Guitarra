# 🎸 Prática de Violão & Guitarra — 90 Dias

App pessoal de estudo com plano de 90 dias, riffs, exercícios técnicos, escalas e acordes.

## 🌐 Acesso

**GitHub Pages:** https://lcarli.github.io/Pratica-Violao-Guitarra/

> ⚠️ **Nota sobre privacidade:** GitHub Pages em conta pessoal gratuita expõe o site publicamente, mesmo se o repositório for privado. Para Pages com acesso restrito, é preciso ter GitHub Pro/Team/Enterprise. Como o conteúdo aqui é só material de estudo público (cifras, escalas, exercícios — coisa que tem em qualquer site de música), deixar o repo público resolve.

**Local (sem rede):** clone o repo e abra `index.html` no navegador.

```bash
git clone https://github.com/lcarli/Pratica-Violao-Guitarra.git
cd Pratica-Violao-Guitarra
# abra index.html no navegador
```

## 📚 O que tem

| Aba | Conteúdo |
|---|---|
| 🎵 **90 Músicas** | Plano progressivo de 3 meses. Acordes borrados (tire de ouvido!). |
| 🎸 **Riffs & Solos** | 25 riffs icônicos em ASCII tab, do Smoke on the Water ao Pride and Joy. |
| 🎯 **Exercícios** | 18 drills técnicos: cromática, hammer-on, bends, vibrato, pestana. |
| 🎼 **Escalas** | Diagramas SVG do braço — pentatônicas, blues, maior, menor, modos. |
| 📖 **Acordes** | Referência visual dos acordes mais usados, com indicação de dedos. |
| 📅 **Rotina** | Sugestão de prática diária de ~50 min. |

## ✨ Features

- Progresso salvo no `localStorage` (persiste entre visitas)
- Acordes/tabs **borrados** por padrão — clique pra revelar individual ou tudo de uma vez
- Diagramas de braço gerados em SVG (alta qualidade em qualquer zoom)
- Tudo em um único arquivo HTML (sem build, sem dependências)

## 🛠️ Stack

HTML + CSS + Vanilla JS. Zero dependências. Roda em qualquer navegador moderno.
